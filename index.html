Hand

Post-WIMP user interface suitable for chorded keyboards.

Download this project as a .zip file Download this project as a tar.gz file

The underlying idea is a different usage paradigm for a wide spectrum of computer types, from small mobile phones and computers to the full blown desktop PCs. This concept perfectly complements and suites the major current and foreseeable future developments in the computer industry.

The new usage paradigm is to a greater extent than the current standard usage paradigm independent of kind of hardware used as an input device and can be even used with the common devices like keyboard, mouse, touchscreen, camera. But the full potential is tapped with data gloves and new kinds of devices.

Thus, the software is the differentiator between classical and the new Hand device.
